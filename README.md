# Cognito AppSync Codegen Authentication (WIP)

A [fetch client](https://the-guild.dev/graphql/codegen/docs/config-reference/schema-field#customfetch) for [GraphQL Code Generator](https://the-guild.dev/graphql/codegen) to allow authentication with [AWS Cognito](https://aws.amazon.com/appsync/) which will then allow you to perform introspection.
